---
title: "Quickstart"
description: "Run kbot against a live Kahoot game in minutes — no account required if you have the quiz UUID."
---

There are two ways to play:

- **By quiz UUID** — fastest path, no Kahoot account needed
- **By quiz name** — requires a Kahoot account (email + password)

Pick the tab that matches what you have available.

<Tabs>
  <Tab title="By quiz UUID (no account)">
    This is the fastest path. You need the game pin and the quiz UUID — no Kahoot account required.

    <Steps>
      <Step title="Get the game pin">
        Ask the host for the **game pin** displayed on the Kahoot lobby screen. It is a 6–8 digit number.
      </Step>

      <Step title="Get the quiz UUID">
        The quiz UUID is a string in the format `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`. You can find it in:

        - The URL when viewing the quiz on [create.kahoot.it](https://create.kahoot.it): `https://create.kahoot.it/details/<uuid>`
        - The URL in the game host's browser when they launch the quiz

        <Note>
          Not all quizzes are publicly accessible. If the quiz is private, you need the host's Kahoot account or must use the search-by-name path instead.
        </Note>
      </Step>

      <Step title="Choose a nickname">
        Pick any nickname you want to appear as in the game lobby.
      </Step>

      <Step title="Run kbot">
        ```bash
        python3.9 kbot -n <nickname> -p <pin> -i <quizID>
        ```

        For example:

        ```bash
        python3.9 kbot -n Player1 -p 123456 -i 550fa14b-4b7e-46ac-a239-e8b6d9dc7a89
        ```
      </Step>

      <Step title="Watch kbot answer">
        kbot will join the game and print each question and the correct answer to your terminal as the game progresses:

        ```
        Joined game as Player1
        Question 1: What is the capital of France?
        Correct answer: Paris (index 2)
        Submitted answer 2
        Question 2: What is 2 + 2?
        Correct answer: 4 (index 0)
        Submitted answer 0
        ```

        Correct answers are printed in green. No further interaction is needed — kbot handles the rest automatically.
      </Step>
    </Steps>
  </Tab>

  <Tab title="By quiz name (requires account)">
    Use this path when you know the quiz's name but not its UUID. kbot will search the Kahoot API on your behalf.

    <Steps>
      <Step title="Get the game pin">
        Ask the host for the **game pin** displayed on the Kahoot lobby screen. It is a 6–8 digit number.
      </Step>

      <Step title="Find the quiz name">
        Get the exact name of the quiz as it appears on Kahoot. Partial matches are supported — kbot will return the closest results. You can control how many results are returned with `-m`.
      </Step>

      <Step title="Have your Kahoot credentials ready">
        You need the email address and password for a [create.kahoot.it](https://create.kahoot.it) account.

        <Warning>
          Your credentials are passed as CLI arguments and may be visible in your shell history. Consider using a dedicated Kahoot account rather than your primary one.
        </Warning>
      </Step>

      <Step title="Run kbot">
        ```bash
        python3.9 kbot -n <nickname> -p <pin> -q "<quiz name>" -e <email> -a <password>
        ```

        For example:

        ```bash
        python3.9 kbot -n Player1 -p 123456 -q "World Capitals" -e user@example.com -a mypassword
        ```
      </Step>

      <Step title="Select the quiz">
        If multiple quizzes match your search, kbot will list them and prompt you to pick one:

        ```
        Found 3 quizzes matching "World Capitals":
        [0] World Capitals (550fa14b-4b7e-46ac-a239-e8b6d9dc7a89)
        [1] World Capitals Quiz (81be3450-2af2-4b7f-b0c1-3cde1f2a9b44)
        [2] World Capitals - Advanced (f4a12c30-9d82-4e67-b1f7-dc8aef3b2c01)
        Select quiz [0-2]: 
        ```

        Enter the number for the quiz you want, then kbot joins the game and begins answering automatically.
      </Step>

      <Step title="Watch kbot answer">
        kbot will join the game and print each question and the correct answer to your terminal:

        ```
        Joined game as Player1
        Question 1: What is the capital of France?
        Correct answer: Paris (index 2)
        Submitted answer 2
        Question 2: What is the largest country by area?
        Correct answer: Russia (index 1)
        Submitted answer 1
        ```

        Correct answers are printed in green. kbot handles the rest automatically.
      </Step>
    </Steps>
  </Tab>
</Tabs>
